## Obj 3 결과: 다양성 ~ Sentinel DHI — ★ 혹형(hump), 선형 아님
- **선형 가정 시**: Hill q1 ~ DHI누적 +0.12 (p=0.032) — 약함, 오해 소지.
- **비선형(다항식·GAM) 재검정 → 단봉(hump)**: **Hill q1·q2 ~ DHI누적 = inverted-U**
  (다항식 2차 p<0.001, GAM EDF 3.2 p<0.001). **중간 생산성(z≈−0.6)에서 다양성 최대.**
- → 고전적 **생산성–다양성 혹형**을 Sentinel-2 DHI 고해상 plot 규모에서 검출. 선형 +0.12는
  혹형의 상승부만 본 착시였음. **Obj 3의 핵심 결과 = 혹형** (그림 **L03**).
- 검증: 교란강도·구조·분광다양성은 선형/단조(중간교란가설 미지지). 상세 `nonlinear.csv`,`gam.csv`.
- minimum/variation DHI는 비유의.
- **방법 교훈**: 평균·선형 모델은 생태 관계의 혹형을 놓침 — 다항식/GAM 필수. (그림 L01·L02·L03)

## Obj 2 확장 — 강도(severity) + recency (연속, robust)

**Robust 교란 재정의** (`62_disturbance_robust.py`): 분광 교란은 NBR 최대하락이
(a) 절대 >0.10, (b) plot 자체 변동성 대비 >3×MAD, (c) **지속**(하락 후 회복 안 됨)일 때만.
→ 잡음 −0.08 이진(371개 오탐) → **robust 50개**. 총 교란 182(산불84·손실105·분광50).
- **severity** = NBR 하락폭 + Hansen 손실비율 (연속, 교란만 >0)
- **recency** = 2025 − 교란연도 (중앙 8년, 0–38년)

**분석** (`52_severity_recency.py`), cov≥0.9, 128 교란:
- (A) 강도 주효과 / (B) recency 주효과 → **둘 다 비유의** (domain/site 통제 후)
  - ※ H02 raw 산점도는 diversity↓ vs recency(r=−0.40)지만 종조성이 **pooled(시간불변)**
    이라 회복 궤적 아님 + 지역 혼동 → 모델에선 비유의. **수준이 아니라 관계가 바뀜.**
- (C)(D) **조절효과가 핵심** (H03):

| 조절 | 관계 | 상호작용 | p |
|---|---|---|---|
| **× recency** | Nestedness ~ LAI | **+0.48** | 0.0004 |
| × recency | Hill q1/q2 ~ EVI | **−0.52/−0.57** | 0.002–0.005 |
| × recency | Hill q1/q2 ~ SAVI | −0.48/−0.52 | 0.007–0.01 |
| **× severity** | Turnover ~ Deep_Gap | +0.12 | 0.0005 |
| × severity | Hill q1/q2 ~ Deep_Gap | +0.11/+0.09 | 0.0009–0.006 |
| × severity | Nestedness ~ DHI누적 | −0.13 | 0.002 |

**해석**: 분광–알파 커플링(EVI/SAVI~Hill)은 **갓 교란된 곳에서 가장 강하고 시간이 지나며 약해짐**
(recency 음의 상호작용) = 교란이 RS–종 관계를 바꾸고 그 변형이 **회복과 함께 사라짐**(Obj2 시간동태 지지).
구조–다양성 커플링(Deep_Gap~)은 **교란 강도가 클수록 강화**(severity 양의 상호작용).
그림: **H01 강도, H02 recency, H03 조절효과**.

## Obj 2 — BACI (Before-After-Control-Impact) ★ 주 분석

**문제 인식**: 위 분석은 종조성을 pooled(시간불변)로 썼는데 Obj2는 시간 동태 → "회복" 해석 불가.
→ **종조성도 시간분해**해서 정공법으로.

**방법** (`70_baci_diversity.py`, `71_baci_models.py`):
- 원자료(개체 vst_apparentindividual ⨝ 종 mappingandtagging) → 생존 목본 개체를 **교란 전/후 기간별 pool**
  → Hill q1/q2 + coverage. **구간 pooling으로 coverage 0.99/1.00** (effort 문제 해결)
- 교란 plot 79 / 대조 234 (전·후 조사 둘 다 있는 313 plot)
- 분할연도: 교란=교란연도, 대조=사이트 중앙 교란연도. ΔDiv = 후−전.

**결과** (정직하게):
| 검정 | 결과 |
|---|---|
| (A) BACI: ΔRichness ~ impact | **−1.07종 (p=0.051)** — 교란이 종 ~1개 추가 손실 |
| (A) ΔHill q1/q2 ~ impact | 비유의 — **우점구조는 회복력 있음** |
| (B) ΔDiv ~ severity | 비유의 (교란 n=79 작음) |
| (C) ΔHill q1 ~ **ΔLAI** | r=−0.12 (p=0.057) — RS 변화가 다양성 변화 약하게 추적 |

**핵심 메시지**: 제대로(시간분해) 보면 **교란 효과는 실재하나 modest** — 희귀종 손실(richness↓)이고
우점 다양성(Hill)은 회복력. 앞의 pooled "조절효과"는 시간불변 응답의 한계가 일부 반영된 것 →
**BACI가 더 보수적·정직한 Obj2 결론.** 그림 I01(BACI), I02(dose), I03(변화-결합).

## Choi et al. 2023 (J. Ecology) 프레임워크 적용 (`90_choi2023_structure.py`)

다중시기 LiDAR로 교란 영향 + 초기 복잡도 완충 검정. 218 교란 plot(press 103, pulse 115).
press=곤충/병해/만성고사, pulse=산불/풍해/홍수/벌채(급성). pre/post(교란연도 분할) ΔLiDAR.

**(1) press vs pulse 구조변화 (J01)** — LiDAR가 교란체제 구분(Choi 핵심 지지):
| ΔLiDAR | press(곤충) | pulse(산불/풍해) | p |
|---|---|---|---|
| ΔLAI | **−0.18**(엽면적 손실) | **+0.22**(재생) | 0.000 |
| ΔDeep_Gap | ~0 | −0.05(닫힘) | 0.000 |
| ΔCanopy_Ht | −0.61 | −0.14 | 0.089 |
→ press는 고사로 엽면적·수고 감소, pulse는 교란후 재생으로 엽면적 증가·gap 닫힘.

**(2) 초기 복잡도 완충? (J02, Choi 핵심가설)**:
| 영향 | ~ 초기복잡도(pre rugosity) |
|---|---|
| ΔCanopy_Ht | **r=−0.30 (p<0.001)** |
| ΔFHD | r=−0.26 (p<0.001) |
| **ΔRichness** | **r=−0.06 (p=0.53, 무관)** |

→ 초기 복잡도가 구조변화와 강한 음의 상관. **단 평균회귀(RTM) 의심** → 엄밀 통제(아래).

**(3) 평균회귀 통제 — complexity offset 엄밀 재검정 (`91_complexity_rtm.py`, J04)**:
RTM은 교란·대조 양쪽에 작용 → 대조군 기울기를 baseline으로, **impact×complexity 상호작용**이
진짜 완충. 추가로 Δ를 pre 대신 **(pre+post)평균**에 회귀(수학적 커플링 제거). 546 plot(교란218/대조328).

| ΔCanopy_Ht | 값 |
|---|---|
| naive(교란,pre) | r=−0.30 (p<0.001) ← 편향 |
| **대조군 RTM baseline** | **r=−0.26 (p<0.001)** ← 대조군도 같은 기울기! |
| (A) impact×complexity 상호작용 | −0.27 (**p=0.14, 무의미**) |
| (B) 커플링제거(mean) | −0.03 (**p=0.86, ~0**) |
ΔFHD·ΔRichness도 동일 — 상호작용/커플링제거 모두 비유의.

**전체 구조지표 RTM 통제 (impact×complexity 상호작용 + 커플링제거, 8지표)**:
| 지표 | 상호작용 p | 판정 | 방향 |
|---|---|---|---|
| Canopy_Ht, Max_Ht, LAI, FHD, Vert_SD | 0.13~0.95 | ❌ RTM 아티팩트 | naive −0.30은 RTM(대조군 평행) |
| **Deep_Gap (갭 fraction)** | **0.000** | ✅ **진짜 완충** | 대조 기울기 +0.045→교란 +0.004 = 복잡도가 갭형성 억제 |
| **Gini (구조 불평등)** | **0.003** | ✅ **진짜 증폭** | 대조 +0.001→교란 +0.014 = 복잡도가 이질성 증가 |
| Richness, Hill q1 | ns | ❌ 없음 | 종조성 무관 |

→ **정정: "전부 RTM"은 과했음.** 수고·엽면적·FHD는 RTM이나, **Deep_Gap·Gini는 RTM 통제 후에도
교란 특이적 복잡도 효과가 유의**. Choi 2023이 초점 둔 지표(**DGF·Gini**)에서 정확히 완충/증폭이 재현됨:
복잡한 캐노피는 교란 후 **갭을 덜 만들고(완충)** 구조 **이질성은 더 커짐(증폭)**.
- **단, 종 richness는 복잡도가 못 막음** (RTM 통제 후에도 무관) → 구조 회복력 ≠ 조성 보호.

**계산 방법** (그림 J06 워크스루):
1. plot별 LiDAR 지표를 **교란연도 기준 전/후로 나눠 각 기간 평균** → **Δ = 후평균 − 전평균**
   (예 ABBY_002 Deep_Gap: 전 0.735 → 후 0.293, Δ=−0.442)
2. `Δ지표 ~ impact × z(초기복잡도) + severity + (1|site)` 혼합모델
   - 초기복잡도 = 교란 전 Rugosity. impact = 교란1/대조0.
   - **complexity 주효과** = RTM+자연동태(대조군에도 있음)
   - **impact×complexity 상호작용** = 교란 특이적 완충 (=RTM 제거한 순수 효과)
3. 상호작용 유의 & 교란·대조 기울기 갈라짐 → 진짜(Deep_Gap·Gini). 평행 → RTM(Canopy_Ht).
그림: **J05**(8지표 RTM 통제 결과), **J06**(계산 워크스루 + RTM 통제 도식).

그림: J01(press/pulse), J02(복잡도 완충), J03(복잡도 클래스별).
출처: [Choi et al. 2023, J. Ecology 111:1888](https://besjournals.onlinelibrary.wiley.com/doi/10.1111/1365-2745.14145)

## BACI 민감도 점검 (`72_baci_sensitivity.py`)
8개 사양(대조군 분할연도 4종 × 최소표본 3종 × 교란정의 2종):

| 점검 | 결과 |
|---|---|
| **ΔRich 방향·크기** | **8/8 사양 모두 교란이 −1.0~−1.5종** (완전 일관) |
| **비대응 검정(Mann-Whitney)** | **8/8 모두 p≤0.001** (매우 robust) |
| 최소표본 N≥3/5/10 | site-matched에서 −1.23~−1.30, p=0.02~0.03 (안정) |
| **혼합모델(site random) 유의성** | **대조군 분할연도에 민감** — site-matched=유의(p=0.02), global/fixed=비유의 |

**해석(정직)**: 교란→희귀종 ~1종 손실은 **기술적으로 매우 robust**(모든 사양 −1종, 비대응 p≤0.001)
하지만 **보수적 nested 모델 유의성은 대조군 분할 방식에 의존**. 적절한 설계(대조군을 같은 지역·시기로
매칭=site-matched)에선 유의(−1.26, p=0.02)이나, 임의 분할(global)에선 site 효과에 흡수됨.
→ **"modest하지만 실재하는 richness 손실"** 이라는 결론은 유지되나, 강한 단정은 피함. (절대 효과 robust, 형식적 유의성은 경계)
- 모델 주의: OLS+C(domain)9고정효과+cluster SE는 불안정(부호 반전) → site random / 비대응 검정이 적절.
